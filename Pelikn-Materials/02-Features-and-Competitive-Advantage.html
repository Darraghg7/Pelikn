Technical Architecture

Engineered for reliability
and rapid scale

Modern cloud-native stack with offline resilience. Built to handle multi-venue enterprises and thousands of daily compliance records without performance degradation.

Frontend

  • React 18 + Vite 5
  • Tailwind CSS v3
  • React Router v6
  • PWA via Workbox
  • jsPDF for compliance exports

Backend & Data

  • Supabase (PostgreSQL)
  • Row-Level Security (RLS)
  • UK-based data storage
  • UK GDPR / ICO registered
  • Real-time subscriptions

Deployment

  • Vercel (auto-deploy)
  • Zero-downtime deploys
  • Edge CDN globally
  • Auto-backups daily
  • SSL / HTTPS enforced

Multi-Tenant by Design

Each venue is a fully isolated data silo. Row-Level Security enforced at the database level. No cross-venue data leakage possible. Enterprise-grade data isolation without enterprise complexity.

Offline-First Architecture

Service worker caches all critical assets and data. Staff can log temperature readings, complete checklists and clock in/out with zero network connection. Records sync on reconnect automatically.
